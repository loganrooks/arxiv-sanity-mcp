    Create src/arxiv_mcp/content/service.py with:

    ContentService class following EnrichmentService DI pattern:

    1. __init__(self, session_factory, settings, adapter=None, http_client=None):
       - self.session_factory = session_factory
       - self.settings = settings
       - self.adapter = adapter (MarkerAdapter or MockContentAdapter; lazy init in production)
       - self._http_client = http_client (httpx.AsyncClient, lazy init in production)
       - self._rate_limiter = RateLimiter(1.0 / settings.content_rate_limit) using existing RateLimiter from enrichment/openalex.py

    2. async get_variant(self, arxiv_id: str, variant_type: str) -> dict | None:
       - Query content_variants table for existing cached variant
       - If found, return dict with: arxiv_id, variant_type, content, source_url, backend, backend_version, extraction_method, license_uri, quality_warnings, content_hash, converted_at
       - If not found, return None

    3. async get_or_create_variant(self, arxiv_id: str, preferred_variant: str = "best") -> dict:
       - If preferred_variant == "abstract": fetch Paper.abstract from DB, return as dict with variant_type="abstract", content=paper.abstract, backend=None, extraction_method="metadata". No DB storage needed (abstract is already in papers table).
       - If preferred_variant == "best" or "html": check cache first via get_variant. If cached, return. If not, call _acquire_html(arxiv_id). If HTML acquired, store and return. If not (404), fall through to pdf_markdown.
       - If preferred_variant == "best" (fallthrough) or "pdf_markdown": check cache first. If cached, return. If not, call _acquire_pdf_markdown(arxiv_id). Store and return.
       - If nothing available, return {"error": f"No content variant available for {arxiv_id}"}

    4. async _acquire_html(self, arxiv_id: str) -> dict | None:
       - Call fetch_arxiv_html(arxiv_id, self._get_http_client(), self._rate_limiter)
       - If returns None -> return None (no HTML available)
       - Compute content_hash via hashlib.sha256
       - Store in content_variants with variant_type="html", backend="arxiv_html", extraction_method="direct_fetch", source_url
       - Copy license_uri from Paper record
       - Promote Paper.processing_tier to CONTENT_PARSED if not already
       - Return stored variant as dict

    5. async _acquire_pdf_markdown(self, arxiv_id: str) -> dict | None:
       - Download PDF from f"https://arxiv.org/pdf/{arxiv_id}" to tempfile.NamedTemporaryFile (per pitfall 3 / research open question 3)
       - Rate limit the download request
       - If download fails (404, timeout) -> return None
       - Call self.adapter.convert(temp_path, arxiv_id)
       - Delete temp file after conversion
       - Compute content_hash, store in content_variants with variant_type="pdf_markdown"
       - Copy license_uri from Paper record
       - Promote processing_tier
       - Return stored variant as dict

    6. async _store_variant(self, arxiv_id: str, variant_type: str, content: str, result: ContentConversionResult | None, source_url: str | None) -> dict:
       - Shared storage logic: compute hash, get license_uri from Paper, insert/upsert ContentVariant, promote tier
       - Use pg_insert ON CONFLICT for upsert (same pattern as enrichment)

    7. _get_http_client(self) -> httpx.AsyncClient:
       - Lazy init: if self._http_client is None, create httpx.AsyncClient(follow_redirects=True, timeout=30.0)
       - Return self._http_client

    8. async list_variants(self, arxiv_id: str) -> list[dict]:
       - Return all cached variants for a paper (variant_type + converted_at), without full content (for resource listing)

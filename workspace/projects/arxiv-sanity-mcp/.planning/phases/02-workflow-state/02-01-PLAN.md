  <name>Task 2: Alembic migration 002 for workflow tables</name>
  <files>alembic/versions/002_workflow_tables.py</files>
  <action>
    Hand-write Alembic migration 002 (not autogenerated, per Phase 1 convention). Migration creates 5 tables:

    1. **collections** table:
       - id SERIAL PK
       - slug VARCHAR(128) UNIQUE NOT NULL
       - name VARCHAR(256) NOT NULL
       - is_archived BOOLEAN DEFAULT FALSE
       - created_at TIMESTAMPTZ NOT NULL
       - updated_at TIMESTAMPTZ NOT NULL

    2. **collection_papers** table:
       - collection_id INTEGER NOT NULL REFERENCES collections(id) ON DELETE CASCADE
       - paper_id VARCHAR(20) NOT NULL REFERENCES papers(arxiv_id) ON DELETE CASCADE
       - source VARCHAR(32) DEFAULT 'manual'
       - added_at TIMESTAMPTZ NOT NULL
       - PRIMARY KEY (collection_id, paper_id)
       - INDEX on paper_id for reverse lookup queries

    3. **triage_states** table:
       - paper_id VARCHAR(20) PK REFERENCES papers(arxiv_id) ON DELETE CASCADE
       - state VARCHAR(20) NOT NULL CHECK (state IN ('shortlisted', 'dismissed', 'read', 'cite-later', 'archived'))
       - updated_at TIMESTAMPTZ NOT NULL
       - INDEX on state for filtering

    4. **triage_log** table:
       - id SERIAL PK
       - paper_id VARCHAR(20) NOT NULL REFERENCES papers(arxiv_id) ON DELETE CASCADE
       - old_state VARCHAR(20) NOT NULL
       - new_state VARCHAR(20) NOT NULL
       - timestamp TIMESTAMPTZ NOT NULL
       - source VARCHAR(32) DEFAULT 'manual'
       - reason TEXT
       - INDEX on (paper_id, timestamp) for audit trail queries

    5. **saved_queries** table:
       - id SERIAL PK
       - slug VARCHAR(128) UNIQUE NOT NULL
       - name VARCHAR(256) NOT NULL
       - params JSONB NOT NULL
       - run_count INTEGER DEFAULT 0
       - last_run_at TIMESTAMPTZ
       - is_watch BOOLEAN DEFAULT FALSE
       - cadence_hint VARCHAR(20)
       - checkpoint_date DATE
       - last_checked_at TIMESTAMPTZ
       - is_paused BOOLEAN DEFAULT FALSE
       - created_at TIMESTAMPTZ NOT NULL
       - updated_at TIMESTAMPTZ NOT NULL

    Use op.create_table() for each table. Downgrade drops in reverse order. Set revision='002', down_revision='001'. Follow the same file structure as 001_initial_schema.py.
  </action>

    1. Create `src/arxiv_mcp/enrichment/service.py` implementing `EnrichmentService`:

       Constructor: `__init__(self, session_factory, settings, adapter=None)` -- same DI pattern as CollectionService. Default adapter = OpenAlexAdapter(settings).

       Core methods:
       - `async def enrich_paper(self, arxiv_id: str, refresh: bool = False) -> EnrichmentResult`:
         a. Open session, verify paper exists (SELECT Paper WHERE arxiv_id). Raise ValueError if not found.
         b. Check cooldown: SELECT PaperEnrichment WHERE arxiv_id AND last_attempted_at > now - cooldown_days. If within cooldown and not refresh, return early with informational message (don't call API).
         c. Call self.adapter.enrich([arxiv_id]). Get single EnrichmentResult.
         d. Upsert PaperEnrichment: use `from sqlalchemy.dialects.postgresql import insert as pg_insert`. Build INSERT ... ON CONFLICT (arxiv_id) DO UPDATE SET all columns. Map EnrichmentResult fields to ORM columns. Store result.raw_response as openalex_raw. Set enriched_at = now (only if status == success or partial). Set last_attempted_at = now always.
         e. On success/partial: Update Paper.openalex_id (from result). Update Paper.doi ONLY if Paper.doi is currently null and result.doi is not null (never overwrite existing DOI). Update Paper.processing_tier to ProcessingTier.ENRICHED. Update Paper.promotion_reason to "openalex_enrichment".
         f. On not_found/error: Do NOT update Paper columns. Record PaperEnrichment with status and error_detail.
         g. Commit and return the EnrichmentResult.

       - `async def enrich_collection(self, slug: str, refresh: bool = False, dry_run: bool = False) -> dict`:
         a. Query papers in collection (JOIN CollectionPaper, Paper WHERE slug).
         b. Filter out papers within cooldown (unless refresh).
         c. If dry_run: return {"total": len(papers), "to_enrich": len(eligible), "skipped_cooldown": len(skipped)}.
         d. Call self.adapter.enrich(eligible_arxiv_ids) for batch enrichment.
         e. Process results: upsert PaperEnrichment, update Paper columns for each.
         f. Return summary: {"total", "enriched", "not_found", "errors", "skipped_cooldown"}.

       - `async def enrich_search(self, query: str, limit: int = 20, refresh: bool = False) -> dict`:
         a. Run search via SearchService (import and instantiate with same session_factory).
         b. Take first `limit` results.
         c. Same batch enrichment flow as enrich_collection.

       - `async def get_enrichment_status(self, arxiv_id: str) -> PaperEnrichment | None`:
         a. SELECT PaperEnrichment WHERE arxiv_id. Return ORM object or None.

       - `async def get_enrichment_stats(self) -> dict`:
         a. SQL aggregation: COUNT(*) total, COUNT by status, MAX(enriched_at) as last_enrichment.

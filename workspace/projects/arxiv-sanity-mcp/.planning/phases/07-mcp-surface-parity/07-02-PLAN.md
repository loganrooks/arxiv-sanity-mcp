From src/arxiv_mcp/interest/search_augment.py:
```python
class ProfileSearchResponse(BaseModel):
    results: PaginatedResponse[ProfileSearchResult]
    ranker_snapshot: RankerSnapshot | None = None

class ProfileRankingService:
    async def search_papers(self, *, profile_slug=None, page_size=20, **kwargs) -> ProfileSearchResponse: ...
    async def browse_recent(self, *, profile_slug=None, page_size=20, **kwargs) -> ProfileSearchResponse: ...
```

Key behavior of ProfileRankingService:
- profile_slug=None -> delegates to WorkflowSearchService, wraps as ProfileSearchResult with ranking_explanation=None, ranker_snapshot=None
- profile_slug=provided -> over-fetches, re-ranks, adds ranking_explanation, captures ranker_snapshot

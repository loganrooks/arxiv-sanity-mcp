# Milestone v0.1: Core arXiv Discovery MCP

**Status:** Complete and frozen as of 2026-04-16
**Release tag:** `v0.1.0`
**Primary implementation phases:** 1-10
**Current status:** Post-v0.1 inter-milestone exploration

## Overview

`v0.1` delivered the first complete MCP-native arXiv discovery substrate for this project:

- metadata ingestion and lexical search
- workflow state (collections, triage, watches, saved queries)
- explicit interest profiles with structured ranking explanations
- lazy OpenAlex enrichment with provenance
- MCP tools, resources, and prompts validated in real agent workflows
- content normalization with rights-aware serving

The milestone is complete, but the repository has moved directly into spike and deliberation work intended to shape the next milestone rather than extending `v0.1` itself.

## What Shipped

1. **Metadata and search foundation**
   - canonical paper model
   - OAI-PMH + API ingestion
   - fielded lexical search, recent browsing, and seed-based expansion

2. **Workflow substrate**
   - collections
   - triage states
   - saved queries
   - watch/delta tracking

3. **Interest and ranking layer**
   - explicit interest profiles
   - structured ranking explanations
   - inspectable ranking signals

4. **External enrichment and content handling**
   - lazy OpenAlex enrichment
   - content variants
   - provenance and rights-aware access

5. **MCP surface and validation**
   - MCP tools/resources/prompts
   - real agent integration testing
   - release packaging and `v0.1.0` tag

## Completion Notes

- The main implementation roadmap is complete.
- The release tag `v0.1.0` already exists.
- The project is now in a **between-milestones** state: the active work is spike-driven inquiry and deliberation intended to determine the shape of the next milestone.
- This is therefore a **completed milestone with ongoing post-milestone research**, not an active implementation milestone.

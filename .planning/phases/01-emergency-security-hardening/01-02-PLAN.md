ps aux --sort=-%mem | head -25

<task type="auto">
  <name>Task 2: Security posture assessment (AUD-03)</name>
  <files>.planning/research/security-audit.md</files>
  <action>
Run the following commands and capture ALL output. This is a HIGH priority domain -- security findings are the most critical input to Phase 3.

**IMPORTANT: Focus ONLY on Local Address column in ss output.** `0.0.0.0:PORT` = exposed to all interfaces (BAD). `127.0.0.1:PORT` = localhost only (GOOD). `100.93.212.44:PORT` = Tailscale only (ACCEPTABLE). Do NOT confuse Peer Address column with Local Address.

**Services bound to 0.0.0.0 (exposed to ALL interfaces):**
```bash
ss -tlnp | grep '0.0.0.0'
```

**Services bound to [::] (IPv6 all interfaces):**
```bash
ss -tlnp | grep '\[::\]'
```

**Full ss output for analysis:**
```bash
ss -tlnp
```

**Git credential configuration (CRITICAL finding confirmed):**
```bash
git config --global credential.helper
ls -la ~/.git-credentials 2>/dev/null
```

**Sensitive file permissions:**
```bash
ls -la ~/.env ~/.ssh/id_* ~/.git-credentials 2>/dev/null
stat -c '%a %n' ~/.env ~/.ssh/id_ed25519 ~/.ssh/authorized_keys 2>/dev/null
```

**SSH configuration:**
```bash
cat ~/.ssh/authorized_keys 2>/dev/null
grep -r 'PasswordAuthentication\|PermitRootLogin' /etc/ssh/sshd_config 2>/dev/null
cat ~/.ssh/config 2>/dev/null
```

**Nginx status:**
```bash
ls /etc/nginx/sites-enabled/ 2>/dev/null
```

**Loginctl linger status:**
```bash
loginctl show-user rookslog 2>/dev/null | grep -i linger
```

**Check for other credential files:**
```bash
find /home/rookslog -maxdepth 3 -name "*.key" -o -name "*.pem" -o -name "credentials*" -o -name ".env*" 2>/dev/null | head -20
```

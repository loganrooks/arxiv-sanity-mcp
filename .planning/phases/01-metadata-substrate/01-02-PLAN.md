Ingestion pipeline: OAI-PMH harvester, arXiv API client, XML parsers, and metadata mapper

Purpose: Enable bulk and incremental harvesting of arXiv paper metadata via OAI-PMH (primary) and targeted retrieval via arXiv API (secondary). This populates the database that search (Plan 03) queries against.

Output: Working ingestion pipeline with CLI commands (`arxiv-mcp harvest bulk`, `arxiv-mcp harvest incremental`, `arxiv-mcp harvest fetch <arxiv_id>`), XML parsers for all three OAI-PMH formats, and a mapper that converts parsed metadata to canonical Paper records.
